📖 Инструкция по использованию веб-тренажера португальского языка
Данный комплекс состоит из трех основных файлов, которые размещаются в вашем облачном аккаунте на сервисе PythonAnywhere. Программа автоматически генерирует тесты с четырьмя случайными вариантами ответов на основе вашей личной базы данных, ведет интервальное повторение ошибок 
и воспроизводит чистый европейский акцент.
